# Parkinson Protein Classification Dataset

## Description

This dataset contains protein sequences associated with Parkinson's disease and control proteins. It was curated for binary classification tasks in bioinformatics and machine learning.

## Dataset Summary

* Total samples: 306 proteins
* Parkinson-associated proteins: 153
* Control proteins: 153

## Data Source

Protein sequences were obtained from the UniProt database and curated for this study.
