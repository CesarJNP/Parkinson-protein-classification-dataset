# Dataset Description

## Overview
This dataset contains protein sequences associated with Parkinson's disease and control proteins. It was designed for binary classification tasks.

## Dataset composition
- Total samples: 306 proteins
- Parkinson-associated proteins: 153 (label = 1)
- Control proteins: 153 (label = 0)

## Data source
Protein sequences were obtained from the UniProt database and curated for this study.

## Curation criteria
- Proteins associated with Parkinson's disease were selected based on database annotations.
- Control proteins were selected to balance the dataset.
- Duplicate and incomplete sequences were removed.

## Labels
- 1 → Parkinson-associated protein
- 0 → Control protein

## Intended use
This dataset is intended for:
- Machine learning classification tasks
- Bioinformatics analysis
- Feature extraction studies

## Notes
This dataset was curated for research purposes and should be used with proper citation.
